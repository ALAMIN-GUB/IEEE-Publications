# Comparative Analysis of ST-Based SRAM Cell Design in CMOS and CNFET Technologies

## Publication Information

* Conference: IEEE QPAIN 2025
* Research Area: SRAM Design, CMOS, CNFET
* Topics: Power Consumption, Delay Analysis, Stability Evaluation

## Abstract

This study compares ST-based SRAM cell architectures implemented using CMOS and CNFET technologies. Key performance metrics including power consumption, delay, and stability are analyzed to evaluate the suitability of each technology for future memory applications.
