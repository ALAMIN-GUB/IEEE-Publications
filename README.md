# Design and Performance Analysis of a Novel 8-bit Reversible ALU Using 16nm CMOS Technology

## Publication Information

* Conference: IEEE QPAIN 2025
* Research Area: VLSI Design, CMOS Technology
* Topics: Reversible Logic, ALU Design, Low-Power Circuits

## Abstract

This research presents the design and performance evaluation of a novel 8-bit reversible Arithmetic Logic Unit (ALU) implemented using 16nm CMOS technology. The proposed architecture focuses on reducing power consumption while maintaining computational efficiency for modern VLSI applications.
